# Adventure Personality Quiz

An interactive web application that determines a user's **adventure personality type** based on their responses to a series of questions. The quiz dynamically evaluates answers and displays a personality result based on scoring logic.

This project demonstrates **frontend development, interactive UI design, and dynamic JavaScript logic**.

## How It Works

1. The user starts the quiz.
2. Each question presents selectable answer options.
3. Each option contributes to one or more personality score categories.
4. After the final question, the application calculates the highest scoring category.
5. The user is shown their final **adventure personality result**.

Example personality types:
- Explorer
- Planner
- Thrill Seeker
- Nature Lover
